# 주요 명령어

## 개발
```bash
npm install        # 의존성 설치
npm run dev        # 개발 서버 실행 (http://localhost:3000)
npm run build      # 프로덕션 빌드
npm start          # 프로덕션 서버 실행
npm run lint       # ESLint 실행
```

## Git
```bash
git status         # 상태 확인
git add .          # 변경사항 스테이징
git commit -m ""   # 커밋
git push           # 푸시
```

## Vercel 배포
```bash
vercel             # Vercel CLI로 배포 (선택사항)
```
- 또는 GitHub에 푸시하면 Vercel이 자동 배포
